# ssr-demo

examine [vue-hackernews-2.0][https://github.com/vuejs/vue-hackernews-2.0]'s demo

## Install
``` shell
$npm i
```

## Run dev
``` shell
$npm run dev
# open browser http://127.0.0.1:8089
```

## Run
``` shell
$npm run build
$npm run start
# open browser http://127.0.0.1:8089
```
